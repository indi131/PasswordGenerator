# Password Generator

> Красивый и удобный генератор паролей для Google Chrome

[![Chrome Web Store](https://img.shields.io/badge/Chrome-Extension-4285F4?logo=googlechrome&logoColor=white)](https://github.com/indi131/PasswordGenerator)
[![Version](https://img.shields.io/badge/version-1.0.0-6366f1)]()

---

## Возможности

| Функция | Описание |
|---------|----------|
| **Гибкая настройка** | Верхний регистр, нижний регистр, цифры, символы |
| **Длина пароля** | От 4 до 64 символов через слайдер или ввод |
| **Оценка надёжности** | 5 уровней: от «Плохой» до «Невероятный» с визуальной шкалой |
| **Автокопирование** | Автоматическое копирование пароля в буфер |
| **История паролей** | Последние 20 паролей с временем и сайтом |
| **Привязка к сайту** | Для каждого сайта сохраняется свой пароль |
| **Тёмная тема** | Переключение светлой/тёмной темы (следит за системой) |
| **Криптостойкость** | Генерация через `crypto.getRandomValues()` |
| **Копирование** | Один клик — пароль в буфере |

## Установка

### 1. Скачать готовое расширение

[**Скачать Password Generator v1.0.0 (.crx)**](dist/password-generator.crx)

### 2. Установить в Chrome

1. Откройте `chrome://extensions/`
2. Включите **Режим разработчика** (правый верхний угол)
3. Перетащите `.crx` файл в окно браузера
4. Нажмите **Добавить расширение**

---

## Сборка из исходников

Требуется Node.js.

```powershell
# Установить зависимости
npm install

# Собрать расширение
.\build.ps1
```

Готовый `.crx` появится в папке `dist/`.

---

## Использование

1. Нажмите на иконку расширения на панели инструментов
2. Настройте состав пароля (буквы, цифры, символы)
3. Выберите длину
4. Пароль сгенерируется автоматически
5. Нажмите **КОПИРОВАТЬ** или включите автокопирование

> Пароль автоматически запоминается для текущего сайта. При следующем открытии расширения на том же сайте будет показан сохранённый пароль.

- Manifest V3
- Vanilla JS
- CSS Custom Properties (темизация)
- Chrome `activeTab` API
- `crypto.getRandomValues()` — криптостойкая генерация
